# React + TypeScript + Vite + OpenLayers

This template provides a minimal setup to get React working in Vite with HMR, OpenLayers and some ESLint rules.

# Getting started

- Install [nvm](https://github.com/nvm-sh/nvm) if you don't have it already
- Run `nvm use` to install node 20
- Run `yarn install` to install the dependencies

# Running

- Run `yarn dev` in the root folder to start the dev server
